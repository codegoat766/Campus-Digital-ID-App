# Campus Digital ID DApp

Blockchain-based Student Identity System built on Algorand TestNet.

## Features
- NFT-based Student ID
- University Minting
- Student Opt-In
- On-chain Verification
- Modern React UI
- Pera Wallet Integration

## Tech Stack
- React
- Algorand SDK
- Pera Wallet
- IPFS (Pinata)

## How to Run

npm install
npm start
